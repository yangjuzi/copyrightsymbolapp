🔧 Troubleshooting

Option Key Not Working?

  • • Check System Preferences → Keyboard → Input Sources
  • • Ensure US keyboard layout is selected
  • • Try Character Viewer as alternative

Wrong Symbol Appears?

  • • Your keyboard layout might be different
  • • Check keyboard viewer in menu bar
  • • Try Option + Shift + G on some layouts
